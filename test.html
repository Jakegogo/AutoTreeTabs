新功能说明

🌳 美化树形界面

  • ASCII 树形符号: 使用 │ ├─ └─ 等符号显示清晰的树形结构
  • 当前标签页高亮: 当前活动的标签页会用黄色背景高亮显示
  • 自动滚动: 打开弹窗时自动滚动到当前标签页位置
  • 声音状态显示: 播放声音的标签页显示 ♪ 图标，静音显示 × 图标

🎯 智能标签页切换

  • 自动切换: 关闭当前标签页时自动打开合适的下一个标签页
  • 优先级规则:
    1. 优先切换到前一个同级标签页
    2. 没有同级时切换到父节点
    3. 根节点时切换到左侧的同层节点
  • 智能判断: 只在用户主动关闭标签页时生效，窗口关闭时不触发

使用说明

  1. 安装插件: 在 Chrome 扩展程序页面加载插件文件夹
  2. 打开链接: 点击上方的测试链接
  3. 查看树结构: 点击浏览器工具栏中的插件图标
  4. 测试新功能:
    • 观察美化的ASCII树形结构
    • 注意当前标签页的高亮显示
    • 测试自动滚动到当前标签页功能
    • 点击树节点选择标签页
    • 点击父节点选择所有子标签页
    • 悬停显示关闭按钮并测试关闭功能
    • 关闭父节点会关闭所有子节点
